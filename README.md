# webpack base configuration

A tiny project, which includes a base webpack configuration.

* sass compiler
* ES6 babel compiler
* .ejs (Embedded JavaScript templating)

Created on macOS 10.13.6 via [VirtualBox 5.2.12](https://virtualbox.org/)
